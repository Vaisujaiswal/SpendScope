# FAQ

## 1. How does SpendScope work?

SpendScope analyzes your AI tool subscriptions, plans, seat counts, and monthly spend to identify possible savings opportunities and optimization recommendations.

---

## 2. Does the tool require login or signup?

No. Users can generate audits instantly without creating an account. Email capture happens only after value is shown.

---

## 3. Which AI platforms are supported?

SpendScope currently supports:
- ChatGPT
- Claude
- Cursor
- GitHub Copilot
- Gemini
- OpenAI API
- Anthropic API
- Windsurf

---

## 4. Are the savings estimates AI-generated?

No. The core audit engine uses deterministic pricing logic and benchmark rules instead of AI-generated financial recommendations.

AI is only used for generating personalized audit summaries.

---

## 5. Why would Credex care about this tool?

Credex helps companies access discounted AI infrastructure credits. SpendScope helps identify companies that may be overspending and could benefit from alternative pricing or infrastructure optimization.
